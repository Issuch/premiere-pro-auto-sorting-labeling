# premiere-pro-auto-sorting-labeling
EN:
This plugin allows you to sort and label your Premiere Pro files in your project by file type and name. There's a manual and automatic mode that monitors file changes and automatically sorts the file after a short delay!

This plugin is useful if you want to separate titles and still images, as Premiere Pro doesn't allow you to separate their colors. Also, some .mgrot files from plugins that provide ready-made transitions and titles have names like "Text" or "Transition," so you can easily separate .mgrot files.

How to download?
Drag cep-projectsorter to the directory:
C:\Program Files (x86)\Common Files\Adobe\CEP\extensions




RU:
Плагин позволяет сортировать и помечать ваши файлы в Премьере Про в вашем проекте по типу файлов и их наименованию. Есть ручной и автоматический режим, который отслеживает изменение файлов и с небольшой задержкой сам отсортирует файл! 

Плагин полезен, если вы хотите разделять титры и неподвижные изображения, потому что Премьер Про не позволяет разделять их цвет. Также некоторые файлы формата .mgrot из плагинов, которые предоставляют готовые переходы и титры, имеют в себе названия по типу Текст или Переход, поэтому можно будет легко разделять файлы формата .mgrot

Как скачать?
Перетащи cep-projectsorter в путь:
C:\Program Files (x86)\Common Files\Adobe\CEP\extensions
